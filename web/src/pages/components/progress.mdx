## Import

```css
@import "@webtui/css/components/progress.css";
````

## Usage

```html
<progress value="0.5"></progress>
```

## Reference

### Properties

- `--progress-primary`: The fill color
- `--progress-secondary`: The background color
- `--progress-width`: Width of progress bar

```css
#my-progress-bar {
    --progress-primary: var(--foreground0);
    --progress-secondary: var(--background0);
    --progress-width: 24ch;
}
```

### Extending

To extend the Progress stylesheet, define a CSS rule on the `components` layer

```css
@layer components {
    progress, [is-~="progress"] {
        &[variant-="accent"] {
            --progress-primary: orange;
            --progress-secondary: white;
        }
    }
}
```

### Scope

```css
progress, [is-~="progress"] { /* ... */ }
```
